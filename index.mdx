Blevins Holdings Newsroom is a Next.js starter theme for building a fully featured newsroom or press hub. Backed by the Prezly SDK, it handles content delivery, category browsing, media galleries, multilingual routing, and Algolia-powered search — so you can focus on your brand's story rather than infrastructure.

## Get started in three steps

<Steps>
  <Step title="Clone and install">
    Clone the repository and install dependencies with npm.

    ```bash
    git clone https://github.com/bblevins721/blevins-holdings-newsroom
    cd blevins-holdings-newsroom
    npm install
    ```
  </Step>
  <Step title="Configure your credentials">
    Copy the example environment file and fill in your Prezly API key, newsroom UUID, and Algolia API key.

    ```bash
    cp .env.example .env.local
    ```
  </Step>
  <Step title="Run the development server">
    Start the Next.js development server and open the newsroom in your browser.

    ```bash
    npm run dev
    ```

    Navigate to `http://localhost:3000` to see your newsroom live.
  </Step>
</Steps>

<Note>
  If you don't have a Prezly account yet, you can use one of the sample newsroom tokens provided in the [configuration guide](/configuration) to explore the theme with real content.
</Note>

This guide walks you through everything you need to get the newsroom running on your machine. You will clone the repository, install dependencies, configure environment variables, and start the development server.

<Note>
  **Requirements:** Node.js 18.x and npm 9.x. Run `node -v` and `npm -v` to confirm your versions before you start.
</Note>

## Setup steps

<Steps>
  <Step title="Clone the repository">
    Clone the Blevins Holdings Newsroom repository to your local machine.

    ```bash
    git clone https://github.com/bblevins721/blevins-holdings-newsroom
    cd blevins-holdings-newsroom
    ```
  </Step>
  <Step title="Install dependencies">
    Install all Node.js dependencies using npm.

    ```bash
    npm install
    ```
  </Step>
  <Step title="Copy the environment file">
    Create your local environment file by copying the provided example.

    ```bash
    cp .env.example .env.local
    ```

    This creates a `.env.local` file that Next.js will load automatically. Never commit this file to version control — it contains your API credentials.
  </Step>
  <Step title="Fill in your credentials">
    Open `.env.local` in your editor and set the three environment variables.

    ```bash .env.local
    PREZLY_ACCESS_TOKEN=your-prezly-api-token
    PREZLY_NEWSROOM_UUID=your-newsroom-uuid
    ALGOLIA_API_KEY=your-algolia-search-api-key
    ```

    If you do not have a Prezly account yet, use one of the sample tokens below to explore the theme with real content.
  </Step>
  <Step title="Start the development server">
    Run the Next.js development server with live reload enabled.

    ```bash
    npm run dev
    ```
  </Step>
  <Step title="Open the newsroom in your browser">
    Navigate to `http://localhost:3000` to see your newsroom running with content pulled from Prezly.
  </Step>
</Steps>
